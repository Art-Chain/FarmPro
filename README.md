## Development
1. Install depedencies
```shell
npm install
```
2. Run script
```shell
npm run android
```
or
```shell
npm run ios
```
> [!IMPORTANT]
> Do not use `pnpm` or `yarn`. Those are not working in this project.

## External Dependencies
- check `dependencies` and `devDependecies` field in [this file](./package.json)
